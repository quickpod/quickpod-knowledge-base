# QuickPod Overview

QuickPod lets users launch AI workloads, clusters, services, and serverless endpoints on available infrastructure.

## Core Areas

- Templates define container images and startup behavior.
- Pods and clusters run workloads on selected offers.
- Cluster services expose stable endpoints for running workloads.
- Serverless endpoints route public traffic to cluster services.
- Credits and billing govern workload usage and spend.
